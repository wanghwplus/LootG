# LootG 拾取通知

![LootG Icon](icon.png)

**LootG** 是一个魔兽世界插件，在屏幕中央以精美的滚动动画显示拾取的物品。

## 功能特点

- **滚动拾取显示** - 拾取的物品以流畅的动画向上滚动
- **品质颜色** - 物品按品质颜色显示（普通、优秀、精良、史诗、传说）
- **物品图标** - 在物品名称旁显示物品图标
- **背包数量** - 显示背包中该物品的总数量（仅当数量 > 0 时显示）
- **货币与金币** - 同样支持显示拾取的货币和金币
- **自定义位置** - 拖动锚点可将显示位置移动到屏幕任意位置
- **字体自定义** - 可选择多种字体，调整大小、描边和阴影
- **简洁界面** - 只显示你自己的拾取，不显示队友的

## 安装方法

1. 下载并解压 `LootG` 文件夹
2. 将其放入 `World of Warcraft\_retail_\Interface\AddOns\` 目录
3. 重启魔兽世界或重载界面（`/reload`）

## 使用方法

### 命令

- `/lootg` - 打开设置面板
- `/lootg test` - 显示测试拾取消息

### 设置

通过以下方式访问设置：
- 在聊天框输入 `/lootg`
- 点击小地图旁的插件按钮中的 LootG 图标
- 游戏菜单 → 选项 → 插件 → LootG

### 配置选项

| 设置 | 说明 |
|------|------|
| **锁定位置** | 锁定/解锁锚点以调整位置 |
| **显示图标** | 开关物品图标显示 |
| **X/Y 坐标** | 微调显示位置 |
| **显示持续时间** | 消息显示时长（0.5-10秒） |
| **滚动时间** | 动画速度（0.1-5秒） |
| **渐隐速度** | 淡出时长（0.1-2秒） |
| **字号** | 文字大小（8-48） |
| **字体** | 可选标准、聊天、伤害、任务字体 |
| **字体描边** | 无、细描边、粗描边、单色 |
| **字体阴影** | 启用/禁用文字阴影 |

## 显示格式

```
[图标] 拾取 物品名称 x1 (背包数量)
```

- **图标** - 物品材质（可选）
- **拾取** - 前缀文字
- **物品名称** - 按物品品质着色
- **x1** - 拾取数量
- **(背包数量)** - 背包中总数（仅当 > 0 时显示）
